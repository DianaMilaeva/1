Что делает программа:
Программа показывает пять концертов «Шаманчика :D» на разные даты. Пользователь вводит своё имя, 
выбирает концерт, потом может посмотреть схему зала, забронировать место (нужно указать ряд и место) 
или отменить бронь. Если место уже занято, программа не даст его забронировать. Если бронь пытается отменить другой человек — не получится.
Принципы ООП:
В классе Seat поля спрятаны (приватные), а работать с ними можно только через специальные 
методы Book и Cancel — это инкапсуляция. Есть объекты Seat (место) и Hall (зал), каждый отвечает за свою часть работы.
